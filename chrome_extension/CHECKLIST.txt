📥 安装前准备
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[ ] Chrome浏览器或紫鸟浏览器已安装
[ ] 可以访问亚马逊卖家后台
[ ] 有待上传的商品数据


🔧 第一步: 下载SheetJS库 (必须!)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[ ] 创建 chrome_extension/libs/ 目录
[ ] 访问: https://cdn.sheetjs.com/xlsx-0.20.1/package/dist/xlsx.full.min.js
[ ] 下载并保存为: chrome_extension/libs/xlsx.full.min.js
[ ] 验证文件大小约800KB


🎨 第二步: 创建图标 (可选,不影响功能)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[ ] 在 chrome_extension/icons/ 目录创建:
    [ ] icon16.png  (16x16像素)
    [ ] icon48.png  (48x48像素)
    [ ] icon128.png (128x128像素)

    💡 提示: 暂时没有可以跳过,不影响功能


🚀 第三步: 安装扩展
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[ ] 打开浏览器
[ ] 访问 chrome://extensions/
[ ] 开启右上角 "开发者模式"
[ ] 点击 "加载已解压的扩展程序"
[ ] 选择 chrome_extension 文件夹 (整个文件夹,不是里面的文件)
[ ] 扩展出现在列表中,状态为 "已启用"


✅ 第四步: 验证安装
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[ ] 浏览器工具栏出现插件图标
[ ] 点击图标,弹出插件界面
[ ] F12打开控制台,看到:
    [学习模式] 已加载
    [页面分析器] 已加载
[ ] 控制台输入 window.pageAnalyzer 返回对象


📊 第五步: 准备Excel数据
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[ ] 创建Excel文件 (.xlsx 或 .xls)
[ ] 第一行为字段名:
    必需: title, brand, price
    可选: description, quantity, sku
[ ] 第二行开始为商品数据
[ ] 保存文件

    示例格式:
    ┌────────────────┬──────────┬───────┐
    │ title          │ brand    │ price │
    ├────────────────┼──────────┼───────┤
    │ 蓝牙耳机       │ SoundMax │ 4980  │
    │ 快充线 2米     │ ChargePro│ 1280  │
    └────────────────┴──────────┴───────┘


🎯 第六步: 首次使用 - 页面分析 (推荐)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[ ] 登录亚马逊卖家后台
[ ] 打开 "添加商品" 或 "创建新商品" 页面
[ ] 点击插件图标
[ ] 点击 "📊 分析当前页面所有元素" 按钮
[ ] 查看弹出的分析报告
[ ] 注意 "智能匹配建议" 中的置信度:
    - >80% (绿色) = 高度推荐,可直接使用
    - 50-80% (橙色) = 可能正确,建议测试
    - <50% (红色) = 不确定,需要学习模式


🎓 第七步: 学习模式 (可选,页面分析置信度低时使用)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[ ] 在亚马逊商品上传页面
[ ] 点击插件图标
[ ] 点击 "🎯 教插件识别页面元素" 按钮
[ ] 按提示依次点击:
    [ ] 商品标题输入框
    [ ] 品牌输入框
    [ ] 价格输入框
    [ ] 商品描述输入框
    [ ] 库存数量输入框
    [ ] SKU输入框
[ ] 每次点击后确认
[ ] 看到 "🎉 学习完成!" 提示


📝 第八步: 第一次填写测试
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[ ] 准备只有1-2个商品的测试Excel
[ ] 在亚马逊商品上传页面
[ ] 点击插件图标
[ ] 点击 "选择Excel文件"
[ ] 选择测试Excel文件
[ ] 看到商品数量显示正确
[ ] 点击 "开始自动填写"
[ ] 观察填写过程:
    [ ] 字段逐个填写
    [ ] 有打字动画效果
    [ ] 填写完成显示绿色边框
[ ] 检查填写结果是否正确


🔍 第九步: 检查填写结果
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[ ] 商品标题正确
[ ] 品牌正确
[ ] 价格正确
[ ] 描述正确 (如果有)
[ ] 库存数量正确 (如果有)
[ ] SKU正确 (如果有)
[ ] 所有字段格式正确,无乱码

    ❌ 如果发现问题:
    [ ] 返回第七步,重新运行学习模式
    [ ] 或修改Excel数据格式
    [ ] 或查看控制台错误信息


🚀 第十步: 批量处理
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[ ] 测试成功后
[ ] 准备完整的商品Excel文件
[ ] 勾选 "自动填写下一个商品" (可选)
[ ] 点击 "开始自动填写"
[ ] 观察进度条
[ ] 等待所有商品填写完成
[ ] 人工检查所有商品
[ ] 确认无误后提交到亚马逊


⚙️ 设置选项说明
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ 模拟真人打字速度
   推荐: 开启
   作用: 每个字符间隔50-150ms,更真实

✅ 自动填写下一个商品
   推荐: 测试时关闭,正式使用时开启
   作用: 填完一个自动填下一个(延迟2秒)


🐛 常见问题快速解决
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
问题: 点击插件图标没反应
解决: [ ] 右键插件图标 → "检查弹出内容" → 查看错误
      [ ] 最常见: "XLSX is not defined" → 下载xlsx.full.min.js

问题: Excel读取失败
解决: [ ] 检查文件格式 (.xlsx 或 .xls)
      [ ] 检查第一行是字段名
      [ ] 确保包含必需字段: title, brand, price

问题: 填写时找不到元素
解决: [ ] 运行页面分析,查看元素
      [ ] 运行学习模式手动指定
      [ ] 刷新页面重试

问题: 控制台看不到加载信息
解决: [ ] 刷新页面
      [ ] 重新加载扩展(chrome://extensions/)
      [ ] 检查URL是否匹配亚马逊域名

问题: 图标不显示
解决: [ ] 不影响功能,可以忽略
      [ ] 或创建icon PNG文件后重新加载扩展
