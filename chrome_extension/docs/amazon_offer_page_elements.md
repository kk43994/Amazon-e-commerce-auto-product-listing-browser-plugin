## 特殊说明
1. 带 ⭐ 标记的为重要字段
2. 带 * 标记的为页面必填字段
3. 价格相关字段都有货币符号 JPY¥ 前缀
4. 日期字段都有日历选择器按钮
5. 配送渠道选择会影响后续的配送设置选项
6. FBA（亚马逊配送）被标记为"推荐"选项
7. 产品尺寸和包装尺寸对于FBA配送尤其重要
8. "让我先跳过报价数据"选项可以暂时保存为不可售状态
