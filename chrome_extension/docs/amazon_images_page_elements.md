## 特殊说明

### 图片上传要求
1. **最大图片数量**: 9张
2. **主图片**: 第1个槽位为主图片，其他为附加图片
3. **上传方式**:
   - 点击"上传多个文件"按钮批量选择
   - 拖放文件到上传区域
4. **图片顺序**: 可以在上传完成后调整图片顺序

### 页面特点
1. 相比产品详情页，图片页结构简单
2. 主要是图片上传功能，没有复杂的表单字段
3. 支持批量上传和拖放上传
4. 有明确的图片数量限制（最多9张）

### 开发建议
1. 实现图片上传时需要处理：
   - 文件选择对话框
   - 拖放事件
   - 图片预览
   - 顺序调整
2. 需要验证图片格式和大小限制
3. 主图片位置固定，需要特殊标记
4. 上传进度需要实时更新显示
