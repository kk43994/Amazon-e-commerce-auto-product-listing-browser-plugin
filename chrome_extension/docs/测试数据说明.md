## 🎯 使用方法

### 方法1: 直接使用测试数据

```bash
# 1. 打开Chrome扩展插件
# 2. 点击"选择Excel文件"
# 3. 选择: chrome_extension/test_data_v3.csv
# 4. 点击"开始全自动上传"
```

### 方法2: 修改测试数据

使用Excel或文本编辑器打开 `test_data_v3.csv`:

1. 修改ASIN为真实存在的商品
2. 修改商品信息
3. 修改图片路径（使用本地绝对路径）
4. 保存文件（保持UTF-8编码）

### 方法3: 创建自己的测试数据

使用模板文件创建:

```bash
# 复制模板
cp chrome_extension/product_template_new.csv my_test_data.csv

# 编辑文件，填写商品信息
# 至少填写: asin, title, brand, your_price
```

## ⚠️ 注意事项

### ASIN相关

1. **真实ASIN测试**
   - `B0D9XYDYBJ` 是真实的日本市场商品
   - 可以用于完整流程测试
   - 其他ASIN是模拟的，需要替换为真实ASIN

2. **ASIN搜索功能**
   - 如果ASIN不存在，搜索会失败
   - 建议先取消"自动搜索ASIN"选项
   - 手动导航到表单页面后再填写

### 图片路径

1. **本地路径**
   ```
   Windows: C:/test_images/main.jpg
   Mac/Linux: /Users/username/test_images/main.jpg
   ```

2. **图片准备**
   ```bash
   # 创建测试图片目录
   mkdir test_images

   # 放入测试图片
   # main.jpg, img1.jpg, img2.jpg, ...
   ```

3. **图片要求**
   - 格式: JPG, PNG
   - 大小: 建议 < 5MB
   - 尺寸: 建议 1000x1000px 以上

### 字段缺失处理

- ✅ 空字段会被跳过（不填写）
- ✅ 只填写已提供的字段
- ⚠️ 必填字段（asin, title, brand）不能为空
