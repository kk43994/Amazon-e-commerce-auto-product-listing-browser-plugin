## 特殊说明
1. 带 ⭐ 标记的为重要字段
2. 带 * 标记的为页面必填字段
3. 某些字段（如商品类型、外部产品ID、浏览节点）提交后无法更改
4. "添加更多"链接可以动态增加输入框数量
5. 部分字段有关联的复选框（如"此商品没有品牌名称"）
6. 日期字段带有日历选择器按钮
