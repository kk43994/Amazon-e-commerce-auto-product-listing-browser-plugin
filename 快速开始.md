## ⚡ 3分钟快速测试

### 步骤1: 安装Python依赖

打开命令行，进入项目目录，运行：

```bash
cd C:\Users\zhouk\Desktop\ziniao
pip install requests
```

### 步骤2: 配置测试脚本

用记事本或任何编辑器打开 `test_ziniao_connection.py`，找到这几行：

```python
# 第17行左右 - 紫鸟浏览器路径
ZINIAO_CLIENT_PATH = r"C:\Program Files\ZiNiao\SuperBrowser.exe"

# 第23-25行 - 你的账号信息
ZINIAO_COMPANY = "你的公司名"
ZINIAO_USERNAME = "你的用户名"
ZINIAO_PASSWORD = "你的密码"
```

修改为你的实际信息后保存。

### 步骤3: 运行测试

**重要**: 先关闭紫鸟浏览器（如果正在运行）

然后运行测试脚本：

```bash
python test_ziniao_connection.py
```

### 步骤4: 查看结果

如果看到 "✅ 恭喜！紫鸟浏览器连接测试全部通过！" 就说明成功了！

## ❓ 遇到问题？

### 常见错误速查：

| 错误信息 | 原因 | 解决方法 |
|---------|------|---------|
| 未找到紫鸟浏览器 | 路径配置错误 | 检查 `ZINIAO_CLIENT_PATH` |
| 无法连接 | 浏览器未启动 | 等待更长时间或检查端口 |
| 登录失败 | 账号密码错误 | 确认账号信息是否正确 |
| 没有店铺 | 账号下无店铺 | 在紫鸟客户端创建店铺 |

详细排查步骤请查看 `测试说明.md` 中的"常见问题排查"章节。
