# 亚马逊RPA自动铺货工具

> 基于紫鸟浏览器的亚马逊商品自动上架系统

## 📖 项目简介

本项目为亚马逊卖家提供自动化批量上架商品的RPA工具。通过紫鸟浏览器的防检测能力和 Selenium WebDriver 自动化技术，实现从Excel表格读取商品数据，自动填写商品信息、上传图片，批量完成商品上架。

### 核心功能

✅ **自动控制紫鸟浏览器** - 启动/关闭店铺环境
✅ **Excel数据读取** - 批量导入商品信息
✅ **自动填写表单** - 商品标题、描述、价格等
✅ **批量上传图片** - 主图+最多5张附图
✅ **批量处理** - 队列管理、进度跟踪
✅ **错误处理** - 失败重试、详细日志
✅ **结果报告** - 生成上架结果统计

## 🚀 快速开始

### 1. 环境准备

**必备条件：**
- ✅ Python 3.8+
- ✅ 紫鸟浏览器 (Windows V6.16.0.126+ 或 Mac V6.15.0.44+)
- ✅ 紫鸟账号（需开通WebDriver权限）
- ✅ 亚马逊卖家账号

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 测试连接

**重要**: 第一步是测试紫鸟浏览器连接！

1. 编辑 `test_ziniao_connection.py` 配置你的信息
2. 关闭正在运行的紫鸟浏览器
3. 运行测试：

```bash
python test_ziniao_connection.py
```

详细说明请查看: [测试说明.md](./测试说明.md) 或 [快速开始.md](./快速开始.md)

## 📋 技术架构

### 技术栈

- **Python 3.8+** - 主开发语言
- **Selenium 4.x** - 浏览器自动化
- **Requests** - HTTP通信

- **Pillow** - 图片处理

```

   ├── 店铺环境管理

2. 数据处理器
   ├── Excel 读取解析
   ├── 数据验证清洗
   └── 图片路径处理

3. 亚马逊自动化引擎
   ├── 页面导航
   ├── 表单自动填写
   ├── 图片自动上传
   └── 提交验证

4. 任务调度器
   ├── 批量队列管理
   ├── 并发控制
   ├── 错误重试
   └── 进度跟踪
```

## 📊 工作流程

```
1. 准备Excel表格
   └── 填写商品信息和图片路径

2. 启动紫鸟浏览器
   └── 自动以WebDriver模式启动

3. 登录紫鸟账号
   └── 通过HTTP API验证

4. 选择目标店铺
   └── 启动店铺浏览器环境

5. 批量上架商品
   ├── 读取Excel数据
   ├── 逐个处理商品
   ├── 填写信息
   ├── 上传图片
   └── 提交验证

6. 生成结果报告
   └── 成功/失败统计
```

### 1. 权限申请
- ⚠️ 首次使用必须申请 WebDriver 权限
- 📍 申请地址: https://open.superbrowser.com/
- 💬 联系方式: 添加生态管家微信，备注"紫鸟webdriver"

### 2. 版本要求
- Windows: 紫鸟浏览器 `V6.16.0.126+`
- Mac: 紫鸟浏览器 `V6.15.0.44+`

### 3. 使用限制
- 建议同时运行 2-3 个浏览器实例
- 控制上架频率，避免触发风控
- 每批次建议不超过 100 个商品

### 4. 安全建议
- 不在代码中硬编码账号密码
- 使用配置文件管理敏感信息
- 定期备份重要数据

## 🐛 常见问题

### Q1: 无法启动紫鸟浏览器？
**A**: 检查路径配置，确保紫鸟主进程已关闭

### Q2: 登录失败？
**A**: 确认账号信息正确，确认已开通WebDriver权限

### Q3: 没有店铺？
**A**: 在紫鸟客户端中创建至少一个店铺

### Q4: 连接超时？
**A**: 增加超时时间设置，建议120秒以上

更多问题请查看: [测试说明.md](./测试说明.md#常见问题排查)
