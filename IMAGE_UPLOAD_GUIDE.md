# 图片上传功能使用说明 (v7.0)

本插件支持两种方式上传图片：**本地图片**和**网络图片**。

## 1. 网络图片 (推荐)
直接在 Excel/CSV 表格中填入图片的 URL 地址即可。
- 支持 `http://` 和 `https://` 开头。
- 必须是直链（直接指向 jpg/png 文件），不能是网页地址。
- **示例**: `https://example.com/images/product-1.jpg`

## 2. 本地图片 (需要简单的本地服务器)
由于浏览器安全限制，插件无法直接读取你的硬盘文件（如 `C:\Photos\img.jpg`）。
需要使用 Python 开启一个简单的本地文件服务器来实现上传。

### 第一步：准备图片
1. 创建一个文件夹（例如 `C:\AmazonImages`）。
2. 将所有要上传的产品图片都**复制**到这个文件夹中。
   - **注意**: 插件会忽略路径，只读取文件名。所以请确保所有图片都在这个文件夹的根目录下，不要放在子文件夹里。

### 第二步：开启本地服务器
1. 打开命令提示符 (CMD) 或 PowerShell。
2. 进入图片文件夹：
   ```bash
   cd C:\AmazonImages
   ```
3. 运行 Python HTTP 服务器命令：
   ```bash
   python -m http.server 8000
   ```
   *看到 "Serving HTTP on 0.0.0.0 port 8000 ..." 即表示成功开启。*
   *请保持这个窗口打开，不要关闭。*

### 第三步：填写表格
在 Excel/CSV 表格的图片列中，你可以填入：
- **完整路径**: `C:\AmazonImages\photo1.jpg` (插件会自动提取文件名 `photo1.jpg` 并从本地服务器请求)
- **仅文件名**: `photo1.jpg`

### 常见问题
**Q: 为什么图片上传失败显示 "Fetch failed"?**
A: 请检查：
1. 本地服务器（黑窗口）是否开着？
2. 图片文件名是否完全一致（注意大小写）？
3. 图片是否在服务器运行的文件夹根目录下？
4. 如果是网络图片，链接是否有效（手动复制到浏览器能打开且是图）？

**Q: 为什么只支持 jpg/png?**
A: 亚马逊后台主要接受这些格式，WebP 等其他格式可能会导致上传后不显示。
